## 📖 Overview

Artistry Hub is an AI-powered handmade marketplace designed to connect rural artisans directly with customers across the globe. The platform eliminates intermediaries by providing a modern e-commerce experience with AI-assisted product discovery, multilingual support, secure authentication, and a responsive user interface.

The application is built using modern web technologies and integrates Google's Gemini AI to deliver intelligent recommendations, product descriptions, and conversational assistance.

# 🚀 Getting Started

## Prerequisites

* Node.js 20+
* npm
* Google Gemini API Key

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/artistry-hub.git
```

Move into the project:

```bash
cd artistry-hub
```

Install dependencies:

```bash
npm install
```

---

## Environment Variables

Create a file named **.env.local**

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

You can obtain your Gemini API key from:

https://aistudio.google.com/app/apikey

---

## Run Development Server

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```
